# Mobile Sales Dashboard (Power BI)

## Project Overview
This Power BI dashboard provides insights into mobile sales performance, revenue trends, customer behavior, and regional sales distribution.

## Key Features
- Total Sales Analysis
- Revenue Tracking
- Quantity Sold
- Transaction Analysis
- Regional Performance
- Brand-wise Sales Comparison
- Interactive Filters and Slicers

## Tools Used
- Power BI
- DAX
- Power Query
- Excel Dataset

## Insights
- Top-performing mobile brands
- Best sales regions
- Revenue growth trends
- Customer purchasing patterns
